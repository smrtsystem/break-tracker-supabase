# 🕐 Break Tracker Application

A complete employee break tracking system with vacation management and check-in/out functionality.

## 📋 Features

### 🔐 Authentication
- User login with role-based access (Employee/Admin)
- Admin login with default credentials

### 👤 User Features
- Start/End breaks with multiple break types (Short, Lunch, Personal, Meeting)
- Real-time break duration tracking with overdue warnings
- View active breaks of other employees
- Break history with duration records
- Vacation request management (Add, View, Accept)
- Check In/Out with admin approval workflow

### 🛡️ Admin Features
- **Employee Management:** Add, Edit, Delete employees
- **User Management:** Create, Edit, Delete users with role assignment
- **Department Management:** Add, Edit, Delete departments
- **Role Management:** Create, Edit roles with permissions
- **Break Status:** View all active breaks with overdue indicators
- **Break Log:** View, filter, export break history
- **Vacation Management:** Approve/Reject, Adjust days, Delete vacation requests
- **Check In/Out Management:** Approve/Reject, Delete check-in records
- **Settings:** Configure break durations for Local/Expat employees
